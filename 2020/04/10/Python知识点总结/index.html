    Python知识点总结

    Doint的博客

    Python

    python知识点汇总

    一、Python解释器

    CPython

    当我们从Python官方网站下载并安装好Python 3.x后，我们就直接获得了一个官方版本的解释器：CPython。这个解释器是用C语言开发的，所以叫CPython。在命令行下运行python就是启动CPython解释器。CPython是使用最广的Python解释器。教程的所有代码也都在CPython下执行。

    IPython

    IPython是基于CPython之上的一个交互式解释器，也就是说，IPython只是在交互方式上有所增强，但是执行Python代码的功能和CPython是完全一样的。好比很多国产浏览器虽然外观不同，但内核其实都是调用了IE。CPython用>>>作为提示符，而IPython用In [序号]:作为提示符。

    PyPy

    PyPy是另一个Python解释器，它的目标是执行速度。PyPy采用JIT技术，对Python代码进行动态编译（注意不是解释），所以可以显著提高Python代码的执行速度。绝大部分Python代码都可以在PyPy下运行，但是PyPy和CPython有一些是不同的，这就导致相同的Python代码在两种解释器下执行可能会有不同的结果。如果你的代码要放到PyPy下执行，就需要了解PyPy和CPython的不同点。

    Jython

    Jython是运行在Java平台上的Python解释器，可以直接把Python代码编译成Java字节码执行。

    IronPython

    IronPython和Jython类似，只不过IronPython是运行在微软.Net平台上的Python解释器，可以直接把Python代码编译成.Net的字节码。

    二、Python基础

    数据类型

    整数

    浮点数：由于计算机存储的原因，浮点数运算则可能会有四舍五入的误差

    长整型：末尾加L

    复数：a=x+yj

    字符串：使用'' "" ''' '''来表示，可以使用\来进行转义,\n表示换行，\t表示制表符，r''表示不转义

    列表：list = [1,2,3,4,’n’,’x’]，list[4]=’n’

    元组：使用()标识，但不能二次赋值，使用方式类似于列表

    字典：使用{}标识，使用无序键值对的方式进行记录，dict={‘name’:’tn10d’,’age’:’18’}，dict[‘name’]=’tn10d’

    字节串：b''进行标识，记录的是二进制串

    布尔值：True，False两种

    空值：None

    注意：u''表示以Unicode 格式进行编码

    格式化

    %实现

    举例：’Hi, %s, you have $%d.’ % (‘Michael’, 1000000)；

    %d整数、%f浮点数、%s字符串、%x十六进制整数、%%表示一个%

    format()

    举例：’Hello, {0}, 成绩提升了 {1:.1f}%’.format(‘小明’, 17.125)

    传入的参数依次替换字符串内的占位符{0}、{1}

    f或F

    举例：>>>a = 123123123

    ​ >>>print(f’aaa{a}aaa’)

    可以将变量传入要输出的字符串

    条件判断
    循环

    while一直循环，直到满足条件

    for i in range(n)循环n次

    itertools.product实现多重for循环

    三、函数

    内置函数

    自带的一些参数

    globals()——获取全局变量的字典

    locals()——获取执行本方法所在命名空间内的局部变量的字典

    eval()将字符串类型的代码执行并返回结果

    exec()将自字符串类型的代码执行,返回位None.

    id(o) o是参数，返回一个变量的内存地址

    callable(o)o是参数，看这个变量是不是可调用。 如果o是一个函数名，就会返回True

    *dir() *不带参数时，返回当前范围内的变量、方法和定义的类型列表；带参数时，返回参数的属性、方法列表。

    数字——数据类型相关：bool，int，float，complex
    数字——进制转换相关：bin，oct，hex
    数字——数学运算：abs模，divmod取除和余，min，max，sum，round取n位，pow幂运算

    map()函数应用于每一个可迭代的项，返回的是一个结果list。

    函数的参数

    默认参数：def power(x, n=2)，当只传入x时，默认n为2；当传入x和n时，则n为传入的n；多个默认参数，可以定义参数名 enroll(‘Adam’, ‘M’, city=’Tianjin’)。定义默认参数要牢记一点：默认参数必须指向不变对象！

    可变参数：可变参数定义为 *args,此参数是可变的，可以传入任意数量的组装成tuple（元组）的参数

    关键字参数：**kwargs和可变参数类似，只是将传入参数组装成list

    命名关键字参数：要限制关键字参数的名字，使用方法：def person(name, age, *, city, job)，限制了只能使用city和job

    参数定义的顺序必须是：必选参数、默认参数、可变参数、命名关键字参数和关键字参数。
